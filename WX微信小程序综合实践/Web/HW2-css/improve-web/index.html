Doparion: World Instruction

世界名称与灵感来源

取多巴胺 Dopamine 的前缀 和《海伯利安》 Hyperion 的后缀组合而成的世界名。

  • 多巴胺所指向的「活力」与「有限奖赏」是世界的核心
  • 海伯利安是Seven Pilgrim最终所要到达的朝圣地

作为 Seven Pilgrim 的我们，
成为了带着旧地球人们的希望、梦想与智慧，前往新地球的朝圣者。

世界核心理念

多巴胺，一种神经递质，既是快乐与活力的源泉，也是欲望的根源之一。

活力，意味着

  • 人的可持续
  • 自然的可持续
  • 人与自然交互的可持续

快乐，是人生价值的重要追求之一

然而，过分的快乐，过度膨胀的欲望，正像多巴胺导致行为上瘾，
终将导致能源资源的耗尽，旧地球的毁灭


Seven Pilgrim 作为来到“新地球”上的朝圣者， 以多巴胺为思考原点，
提出了「活力」和「有限奖赏」的核心理念。

坐落在沿海的厦门，以地热能这种脉动不息为主要能源的我们，
将在世界的方方面面体现「活力」，并用积分制做到「有限奖赏」。
